Admin Panels

The admin panel is a general location for viewing the current server settings, user management and testing URLs against project configuration settings.

User management has global rights and project specific rights.

Global permissions:

  • admns - Full access to everything
  • Project Admin - Full access to all projects
  • Project Create - Create new projects, clone other projects with at least read only, full management of created prjects
  • Template Admin - Allow add and delete of templates
  • DNS Admin - Allow add, delete and modify of Proxy and DNS FQDN Mapping

Project Access

  • Admin - Full access to everything in the project
  • Settings - Project settings like enable / disable, description, etc
  • Website Admin - Full access to add, clone, remove, rename websites and all of the settings
  • Website Settings - Website settings only and re-assignment of website for Proxy and DNS FQDN Mapping
  • Files Admin - Full Access to managing files from Managment UI
  • Files Read - Read Access to files from Managment UI
  • Read Only - Read Access to project panels
