Website Settings

The following is a brief description of the website settings panel. The following sections are used for configuring how the website is served to users:

  • Website Name and VHost Preview are just for quick reference and viewing
  • General settings for SSL redirect, maintenance mode, maintenance pages for user or API data response, default document for URI without a filename, custom error pages for user or API responses
  • Mapping rules for API server side execution, static content and static content overrides (files in static location that need to execute server side)
  • Sub mapping where defining a folder path to resolve to another website
