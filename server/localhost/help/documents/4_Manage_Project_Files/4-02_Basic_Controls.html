Basic Controls

This panel is in review for improved functionality and features in future versions. There isn't a lot of functionality at the current time. It is limited to folder creation, file creation with some pre-defined helpers, and folder or file deletion. The lock icons on some folders and files are related website mappings and intention to protect those assets from deletion. If deleting some files like 'config.json', it will deregister the project and websites. It would require a re-creation of the project configuration file. When setting default document, maintenance page, error pages, etc. it will apply lock icons to those files after you have created them. Removing the field values and mappings will remove those locks.

These icons are the current control button in the top menu.
