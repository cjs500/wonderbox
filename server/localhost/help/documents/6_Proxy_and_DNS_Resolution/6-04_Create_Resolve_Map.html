Create Resolve Map

The resolve panel is where you create the resolution mapping to projects and website. See the example with red box below which is made up of the list of current resolution settings and a row at the bottom for creating new resolve mappings.




Start by selecting your project you want to map to.




Fill in all the other fields required (table below) then click the add icon to complete creation of mapping.

Map Type
Proxy Map
Mapping in format of domain + path
DNS FQDN
Mapping in format of domain only
Environment
dev
Development
qa
QA (Test)
stage
Stage (Prod Data Test)
prod
Production Environment
URL
Freeform field for domain or domain + path
Website Target
Select project website, blank list means none created for select project




Once added it will show up in the list and ready to view (as long as the domain name is resolved to server IP).
