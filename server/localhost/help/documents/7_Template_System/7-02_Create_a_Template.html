Create a Template

To create a template, first select the project and click the create template icon.




The following dialog selects the whole project as a template.




The following dialog selects the a website as a template. In this case there is only one website but if there was more you can select multiple. The caveat with a website template is when you have folders and files outside of the website folder that is used like configurations etc. If you select a website template you can exclude important items.




Select 'Create' to complete creating the template.
